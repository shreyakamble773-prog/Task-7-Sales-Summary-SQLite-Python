# Task 7: Basic Sales Summary using SQLite and Python

## Objective
To use SQL inside Python to extract basic sales information such as total quantity sold and total revenue, and visualize it using a bar chart.

## Tools Used
- Python
- SQLite (sqlite3)
- Pandas
- Matplotlib
- Jupyter Notebook

## Steps Performed
1. Created a SQLite database (sales_data.db).
2. Created a sales table.
3. Inserted sample sales data.
4. Executed SQL query using SUM() and GROUP BY.
5. Loaded query results into Pandas DataFrame.
6. Displayed results using print().
7. Created a bar chart to visualize total revenue by product.

## Output
- Sales summary table showing total quantity and revenue.
- Bar chart displaying total revenue by product.
